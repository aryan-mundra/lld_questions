equals() & hashCode() — identity by a field

Override both together, from the same field (id).

@Override
public boolean equals(Object obj) {
    if (this == obj) return true;
    if (!(obj instanceof Seat)) return false;
    Seat other = (Seat) obj;
    return id.equals(other.id);
}

@Override
public int hashCode() {
    return id.hashCode();
}

Defensive copy in constructor

Copy incoming collections so callers can't mutate your state.

public Show(List<Seat> seats) {
    this.seats = new ArrayList<>(seats);
}

synchronized block — one thread at a time

A dedicated final lock guards the critical section.

private final Object lock = new Object();  // key

synchronized (lock) {
    // only ONE thread in here at a time
}

Threads — start ALL, then join ALL

Start every thread first, then wait on each.

Thread t1 = new Thread(new Runnable() {
    public void run() { /* work */ } });
Thread t2 = new Thread(new Runnable() {
    public void run() { /* work */ } });

t1.start();   // start ALL first...
t2.start();
try {
    t1.join(); // ...then wait for ALL
    t2.join();
} catch (InterruptedException e) {
    Thread.currentThread().interrupt();
}
